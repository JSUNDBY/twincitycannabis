What is Twin City Cannabis?

Twin City Cannabis is a free, independent price comparison site for every cannabis dispensary in Minnesota. We pull real menu data from 80 dispensaries multiple times a day so you can compare prices, find deals, and stop overpaying.

No login required. No ads. No affiliation with any dispensary. Just honest data.

Did you know? The same 1g cart can cost $48 at one metro dispensary and $70 at another, 15 minutes away. Same brand, same size, 46% markup.

🔎

How to Use the Site

1. Search for any product

Type a product name, brand, or strain into the search bar. You'll see every dispensary that carries it with side-by-side pricing. The cheapest option is highlighted.

2. Compare prices

Go to Browse to see all products. Key comparison tools:

  • Sort by $/g for flower — compare eighths vs quarters vs 4g jars by real value
  • Sort by $/mg for edibles — see the true cost per milligram of THC
  • Filter by dosage — microdose (1-5mg), standard (6-10mg), or higher
  • Price trend arrows — see if a price dropped or rose recently

3. Find your nearest dispensary

On the Dispensaries page, select "Sort: Nearest to me". Your browser asks for location once, then every card shows distance in miles.

4. Check the TCC Score

Every dispensary gets a 0-100 score based on Pricing, Selection, Service, and Lab Testing. It's algorithmic and never influenced by payment.

Best value flower

Best value edibles
Sort by $/mg. A $4 gummy at 50mg ($0.08/mg) beats a $12 gummy at 50mg ($0.24/mg).
